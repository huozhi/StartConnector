Welcome to "StartConnector" GitHub Pages.

This is a kinect game on Windows based .NET4.0. The game is prepared for football lover who can not enjoying the role of goal-keeper while in room. You could control your body to catch the balls generated from different directions. Or you can use keyboard to control the player.
